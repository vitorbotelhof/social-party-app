# Entre Nós — Multiplayer Philosophy

## Modos

O Entre Nós suporta três ideias de multiplayer:

- 1 celular;
- cada jogador no seu celular;
- híbrido.

## 1 Celular

Melhor para:

- grupos presenciais;
- segredo passado de mão em mão;
- pacing rápido;
- pouca infraestrutura;
- jogos de atuação, dedução e confiança.

Princípio:

o celular circula como objeto social.

## Cada Jogador No Seu Celular

Melhor para:

- votação simultânea;
- respostas privadas;
- grupos maiores;
- jogos com informação individual persistente.

Risco:

- todos olharem para a tela demais.

## Híbrido

Combina:

- celular central;
- ações privadas individuais;
- reveal comum.

Usar apenas quando melhora a experiência social.

## Realtime

Realtime deve parecer invisível.

O usuário não deve perceber:

- sync;
- reconexão;
- listeners;
- ownership;
- conflito de estado.

## Filosofia

Presencial-first.

Mesmo com vários celulares, o objetivo continua sendo interação física.
