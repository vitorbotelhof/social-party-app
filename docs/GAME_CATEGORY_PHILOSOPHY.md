# Entre Nós — Game Category Philosophy

## Tese

Categorias não são gavetas fixas. São portas de entrada para experiências sociais.

O catálogo deve funcionar como Netflix/Spotify/TikTok:

- categorias estáveis;
- tags flexíveis;
- contextos dinâmicos;
- curadorias vivas;
- recomendação por sessão.

## Problema A Evitar

Categorias criativas demais podem confundir.

Categorias mecânicas demais podem não vender emoção.

O sistema ideal combina:

- clareza de produto;
- sabor editorial;
- escalabilidade.

## Categorias Principais

Devem ser:

- poucas;
- estáveis;
- fáceis de entender;
- úteis para decisão.

Elas respondem:

"que tipo de noite esse jogo cria?"

## Tags

Tags são muitas e flexíveis.

Elas respondem:

"que comportamento aparece?"

Um jogo pode ter várias tags.

## Contextos

Contextos são dinâmicos.

Eles respondem:

"quando faz sentido recomendar isso?"

Exemplos:

- início da noite;
- grupo novo;
- grupo íntimo;
- energia baixa;
- depois de jogo pesado;
- barulhento;
- fim de noite.

## Curadorias

Curadorias são fileiras editoriais temporárias ou dinâmicas.

Exemplos:

- para ninguém confiar em ninguém;
- para começar sem pensar;
- para grupos que já se conhecem demais;
- para levantar a sala;
- jogos rápidos para trocar energia.

## Regra De Escala

O sistema deve funcionar com 10 jogos e com 300 jogos.

Se uma categoria não ajuda o usuário a escolher, ela não é categoria: é só copy.
