# ENTRE NÓS — ROADMAP

Entre Nós — Product Roadmap
Vision Evolution Roadmap
1. Objetivo do Roadmap

Este roadmap NÃO existe para:

acumular features
adicionar jogos infinitamente
transformar o produto em uma “super plataforma”
criar um social network tradicional

Este roadmap existe para:

transformar o Entre Nós na principal plataforma de experiências sociais presenciais do Brasil.
2. Visão de Longo Prazo

O Entre Nós NÃO será:

apenas um app de party games
apenas uma coleção de experiências sociais
apenas um multiplayer casual

O Entre Nós deve evoluir para:

um sistema operacional de energia social.
O produto deve:
entender grupos
modular emoções
criar memória coletiva
facilitar intimidade
provocar caos
acelerar conexão humana
3. O Verdadeiro Produto

O produto NÃO entrega:

gameplay.

O produto entrega:

noites memoráveis entre pessoas reais.
O objetivo final é:

que as pessoas pensem:

“a gente precisa abrir o Entre Nós.”

da mesma forma que hoje pensam:

“vamos colocar música”
“vamos pedir comida”
“vamos abrir vinho”
4. Roadmap Philosophy

Toda evolução do produto deve priorizar:

energia social
replayability emocional
memória coletiva
flow social
velocidade de interação
dinâmica humana
presença física
NÃO priorizar:
gamificação vazia
retenção artificial
mecânicas complexas
features sociais genéricas
sistemas de profile tradicionais
5. Fase Atual do Produto

Estado atual:

foundation stage.

Objetivos:

consolidar identidade
estabilizar gameplay
validar loops emocionais
refinar direção de produto
encontrar assinatura emocional
